Launch Quality LLC v49 — Direct Production Deployment

Repo: https://github.com/walednajjar2-salam/jawdah-cloud-v47

1) Push branch to GitHub (repo root)
2) Railway -> Deploy from GitHub -> select jawdah-cloud-v47
3) Ensure Volume exists and is mounted:
   /app/data
4) Variables:
   JAWDAH_HOST=0.0.0.0
   JAWDAH_DATA_DIR=/app/data
   JAWDAH_BACKUP_DIR=/app/data/backups
   JAWDAH_AUTO_BACKUP=1
   JAWDAH_BACKUP_INTERVAL_HOURS=24
   JAWDAH_BACKUP_RETENTION=30
   JAWDAH_STORAGE_WARN_GB=2
   ADMIN_PASSWORD=<secure-password>
5) Generate domain and verify:
   /api/health
   /api/backup/status
   /api/permissions/audit (admin)
6) Smoke flow:
   login -> create client -> create/approve contract -> invoice -> payment -> backup

Start command: python server.py
Health check: /api/health
